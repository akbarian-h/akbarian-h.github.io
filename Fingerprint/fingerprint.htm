Fingerprint Authentication System

Jan 2009

Thanks to this system it will be possible to control access to the pre-defined gates, some of the features of this system are:

A powerful database which can save enter/exit time and events.

Possibility  of sensor networking.

A powerful software with  particular capabilities.

I designed this project under supervision of Prof.Khojandi my teacher in the university. This project is designed to use in BANK MASKAN in Iran, in brief this system uses fingerprint data to give a number to the people and sorts them regarding to the orders and credits.
